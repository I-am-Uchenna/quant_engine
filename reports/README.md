# Reports

**`VALIDATION.pdf`** — model validation report for the Hull-White / G2++ Bermudan
swaption engine. Independent benchmarking against QuantLib, Monte Carlo convergence,
an Andersen-Broadie primal-dual policy bound, G2++ decorrelation, real-FRED
parameterisation, risk and exposure analytics, AAD Greeks, and a documented limitations
section framed against SR 11-7.

- `VALIDATION.pdf` — typeset report (5 pages).
- `VALIDATION.tex` — LaTeX source.
- `VALIDATION.md` — Markdown version (renders on GitHub).

Every figure is reproduced by a script in [`../research`](../research).
